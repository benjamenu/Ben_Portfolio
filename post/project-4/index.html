As you can see below, the data is a matrix of 1797 rows and 64 columns. Because the matrix has 64 columns, this means that our data has 64 features. Each feature represents another dimension in our data. So if our data has 1-dimension, then we can represent our data by plotting the data on a number line. If the data had 2-dimensions we could visualize the data by plotting the data on a scatter plot. Since our data has 64-dimensions, we can not visualize the data to understand how well our machine learning algorithm’s are performing. To get around this we perform a technique call PCA to perform dimensionality reduction on our data

Scale the Data

The first step is to scale our data. Using the standard scaling will scale the data by removing the mean and scaling to the unit variance. The standard score of a sample x is calculated by $z=(x-u)/s$. Where z is the output, x is the input, u is the mean, and s is the standard deviation. This is useful if you have data has different magnitudes.

Finding the Covariance Matrix

Covariance can only be calculated between two variables. The covariance matrix represent covariance values for each pair of values. However, something to note is that the coavariance between the same values is the variance. The values in the covariance matrix show the distribution magnitude and direction of multivariate data in multidimensional space. This tells us how data spread among two dimensions.
